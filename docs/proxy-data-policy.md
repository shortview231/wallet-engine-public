# Proxy Data Policy

## Why proxy data is used

This public repo is designed to show the wallet engine without exposing real financial information. All published example data and demonstration visuals use proxy values.

## Rules

- No real balances are published
- No real merchant names are published
- No real account-linked identifiers are published
- No raw source screenshots with live values are published
- No secrets or automation credentials are published

## Demonstration artifacts

- `examples/` contains proxy datasets that mirror the real structural shape
- `screenshots/` contains recreated demo visuals based on proxy values

## Goal

The public artifacts should preserve system credibility, data shape, and workflow meaning while removing anything sensitive or personally identifying.
